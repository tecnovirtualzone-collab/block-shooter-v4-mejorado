# 📖 Guía de Usuario - Block Shooter V4

## 🎮 Bienvenido a Block Shooter V4

¡Gracias por jugar Block Shooter V4! Este juego arcade te desafía a destruir bloques mientras evitas enemigos y recolectas power-ups para mejorar tu experiencia de juego.

---

## 📱 Cómo Jugar

### Objetivo Principal
Destruye todos los bloques en cada nivel para avanzar. Cuantos más bloques destruyas, más puntos obtendrás. ¡Completa todos los niveles para ganar!

### Controles

#### 🖥️ Teclado (Desktop)
| Tecla | Acción |
|-------|--------|
| **← (Flecha Izquierda)** | Mover plataforma a la izquierda |
| **→ (Flecha Derecha)** | Mover plataforma a la derecha |
| **A** | Mover plataforma a la izquierda |
| **D** | Mover plataforma a la derecha |
| **ESPACIO** | Disparar |
| **P** | Pausar/Continuar el juego |
| **ESC** | Volver al menú principal |

#### 🖱️ Mouse (Desktop)
- **Mover el ratón:** Controla la posición de la plataforma
- **Clic izquierdo:** Dispara proyectiles

#### 📱 Móvil/Tablet (Táctil)
- **Botón Izquierdo:** Mover plataforma a la izquierda
- **Botón Derecho:** Mover plataforma a la derecha
- **Botón Central:** Disparar

---

## 🎯 Mecánicas del Juego

### Plataforma
La plataforma es tu base de operaciones. Mueve la plataforma para posicionarte y disparar a los bloques.

### Bloques
- **Bloques normales (🟥):** Destruibles con un impacto
- **Bloques resistentes (🟨):** Requieren 2 impactos para destruir
- **Enemigos (🟠):** Bloques que se mueven y pueden dañarte
- **Power-ups (⚡/✦/🛡/⏳):** Objetos especiales que mejoran tus habilidades

### Power-ups
Recoge estos objetos para obtener ventajas temporales:

| Icono | Tipo | Efecto | Duración |
|-------|------|--------|----------|
| ⚡ | Multi Shot | Dispara 3 proyectiles simultáneamente | 8 segundos |
| ✦ | Laser | Disparo continuo en línea recta | 8 segundos |
| 🛡 | Shield | Protección temporal contra daños | 8 segundos |
| ⏳ | Time Slow | Reduce la velocidad de los enemigos | 8 segundos |

### Enemigos
Los enemigos son bloques que se mueven de lado a lado. Si chocas con ellos, pierdes una vida.

---

## 🏆 Sistema de Puntuación

Gana puntos destruyendo bloques y enemigos:

| Acción | Puntos |
|--------|--------|
| Destruir bloque normal | 10 puntos |
| Destruir bloque resistente | 20 puntos |
| Destruir enemigo | 50 puntos |
| Recoger power-up | 100 puntos |
| Completar nivel | 500 puntos |
| Multiplicador de combo | x2, x3, x5 |

### Records
El juego guarda tus mejores puntuaciones:
- **Mejor puntuación:** La puntuación más alta que has logrado
- **Top 10 records:** Las 10 mejores puntuaciones de todos los tiempos

## 🎮 Niveles

El juego tiene 10 niveles con dificultad creciente:

### Nivel 1-3: Principiante
- Pocos bloques
- Sin enemigos
- Power-ups raros

### Nivel 4-6: Intermedio
- Más bloques
- Aparecen enemigos
- Power-ups más frecuentes

### Nivel 7-9: Avanzado
- Muchos bloques
- Enemigos rápidos
- Power-ups comunes

### Nivel 10: Experto
- Máxima dificultad
- Todos los tipos de bloques
- Power-ups garantizados
- ¡Victoria!

## 💡 Consejos y Trucos

### Estrategias Básicas
1. **Posiciónate bien:** Mantén la plataforma centrada para cubrir más área
2. **Destruye power-ups primero:** Te dan ventajas importantes
3. **Evita enemigos:** Chocar con ellos cuesta vidas
4. **Usa power-ups estratégicamente:** El láser es excelente para niveles difíciles

### Power-ups Recomendados
- **Multi Shot:** Ideal para niveles con muchos bloques
- **Laser:** Perfecto para enemigos y bloques resistentes
- **Shield:** Útil cuando hay muchos enemigos
- **Time Slow:** Te da más tiempo para reaccionar

### Combinaciones
- **Multi Shot + Laser:** Destruye todo a tu paso
- **Shield + Time Slow:** Sobrevive a situaciones difíciles
- **Multi Shot + Shield:** Ataque y defensa combinados

## 🔧 Solución de Problemas

### Problemas Comunes

**❓ El juego no se carga**
- **Solución:** Verifica tu conexión a internet
- **Solución:** Asegúrate de que tu navegador soporte HTML5 Canvas
- **Solución:** Limpia la caché del navegador (Ctrl+Shift+Del)
- **Solución:** Prueba en otro navegador (Chrome, Firefox, Edge)

**❓ Los controles no responden**
- **Solución:** Haz clic en el canvas para darle foco
- **Solución:** Prueba con diferentes dispositivos de entrada
- **Solución:** Reinicia el juego

**❓ No se escucha el audio**
- **Solución:** Verifica que el navegador permita audio
- **Solución:** Asegúrate de que los altavoces/auriculares estén conectados
- **Solución:** Ajusta el volumen del sistema
- **Solución:** Prueba con otro navegador

**❓ El juego va muy lento**
- **Solución:** Cierra otras pestañas o aplicaciones
- **Solución:** Reduce la calidad gráfica en ajustes
- **Solución:** Reinicia tu navegador

**❓ No se guardan los records**
- **Solución:** Asegúrate de que tu navegador permita almacenamiento local
- **Solución:** Prueba en modo incógnito (pero los records no se guardarán)
- **Solución:** Usa el mismo navegador y dispositivo

## 🎨 Personalización

### Cambiar el Tema
Puedes personalizar la apariencia del juego editando el archivo `style.css`:

```css
/* Cambiar colores principales */
:root {
    --primary-color: #00ffff;  /* Cian */
    --secondary-color: #ff00ff; /* Magenta */
    --background-color: #0a0a1a; /* Azul oscuro */
    --text-color: #ffffff; /* Blanco */
}

/* Cambiar tipografía */
@import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap');
```

### Añadir Nuevos Power-ups
Para añadir nuevos power-ups:

1. **Edita `game.js`:**
   ```javascript
   // Añade nuevo power-up en la clase PowerUp
   this.colors.newPowerUp = '#color';
   this.icons.newPowerUp = '🔥';
   
   // Añade lógica en la clase Player
   this.powerUps.newPowerUp = false;
   
   // Añade efecto en game.js
   player.activatePowerUp('newPowerUp', 500);
   ```

2. **Añade sonido:**
   ```javascript
   // En AudioSystem
   this.sounds.sfx.newPowerUp = new Howl({
       src: ['assets/audio/sfx/new_powerup.mp3'],
       volume: 0.9
   });
   ```

3. **Genera power-ups:**
   ```javascript
   // En Level.generateLevel()
   const isNewPowerUp = Math.random() < 0.05;
   if (isNewPowerUp) {
       this.powerUps.push(new PowerUp(this.game, x - 15, y, 'newPowerUp'));
   }
   ```
